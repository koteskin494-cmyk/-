# Blog Application

Full-stack blog application with React frontend and Node.js/Express backend.

## Features

### Backend
- RESTful API with Express.js
- MongoDB with Mongoose ODM
- JWT Authentication
- File upload with Multer
- Input validation
- Rate limiting
- Security headers
- Error handling

### Frontend
- React with functional components and hooks
- React Router for navigation
- React Query for data fetching
- Context API for state management
- Responsive design
- Form handling

## Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcryptjs
- Multer
- Helmet
- Express Rate Limit

### Frontend
- React 18
- React Router DOM
- Axios
- React Query
- React Hook Form
- Date-fns
